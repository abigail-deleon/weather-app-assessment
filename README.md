Project Overview
A comprehensive weather application built for the AI Engineer Intern Technical Assessment. This application allows users to search for weather information by location, view forecasts, and manaage their search history.

Features Implemented
Tech Assessment 1
    -Location-based weather search (City, Zip Code, GPS Coordinates)
    -Current weather display with detailed information
    5-day weather forcast
    -Current location detection (geolocation)
    -Weather icons and visual indicators
    -Real-time API data (OpenWeatherMap API)

Tech Assessment 2
    -Operations
        -Create: Save weather searches with location and date range
        -Read: View all previous weather queries
        -Update: Modify saved weather records
        -Delete: Remove weather records
    -Input Validation
        -Date range validation
        -Location verification with fuzzy matching
        -Error hadling for invalid inputs
    -API Integration (Optional Features)
        -Weather data from OpenWeatherApp
        -Google Maps integration for location visualization
        -YouTube videos for searched locations
    -Data Export
        -Export data to JSON, CSV, PDF formats
